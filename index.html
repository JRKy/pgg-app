Password Grid Generator

How to Use

This tool generates a secure password grid that you can print and carry with you. Here's how it works:

  1. Enter a seed phrase (or use the random one provided)
  2. Customize the grid size and character sets
  3. Print the grid and keep it in a safe place
  4. To use a password, follow a path through the grid

For example, to create the password "P@ssw0rd":

  • Start at cell A1 (P)
  • Move right 3 cells (@)
  • Move down 2 cells (s)
  • Move right 1 cell (s)
  • Move down 1 cell (w)
  • Move right 2 cells (0)
  • Move down 1 cell (r)
  • Move right 1 cell (d)

Your password is: P@ssw0rd
